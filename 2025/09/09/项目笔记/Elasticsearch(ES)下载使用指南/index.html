иҜҒд№Ұз”ҹжҲҗ пјҡе®үиЈ…ж—¶дјҡиҮӘеҠЁз”ҹжҲҗ TLS/SSL иҜҒд№ҰгҖӮ

Elasticsearch еҗҜеҠЁж—¶й»ҳи®ӨеҗҜз”ЁдәҶ HTTPSпјҲеҠ еҜҶйҖҡдҝЎпјүпјҢеҰӮжһңе°қиҜ•йҖҡиҝҮ HTTPпјҲжҳҺж–ҮйҖҡдҝЎпјүи®ҝй—® ElasticsearchпјҢдјҡеҜјиҮҙдәҶд»ҘдёӢй”ҷиҜҜпјҡ

иҝҷиЎЁжҳҺдҪ зҡ„е®ўжҲ·з«ҜпјҲеҰӮ curl жҲ–жөҸи§ҲеҷЁпјүеҸ‘йҖҒзҡ„жҳҜ HTTP иҜ·жұӮпјҢиҖҢ Elasticsearch й…ҚзҪ®дёәд»…жҺҘеҸ— HTTPS иҜ·жұӮ

ж–№жі• 1пјҡдҪҝз”Ё HTTPS и®ҝй—® Elasticsearch

й»ҳи®Өжғ…еҶөдёӢпјҢElasticsearch дјҡеңЁе®үиЈ…зӣ®еҪ•дёӢз”ҹжҲҗдёҖдёӘиҮӘзӯҫеҗҚиҜҒд№ҰпјҢе№¶еҗҜз”Ё HTTPSгҖӮдҪ еҸҜд»ҘйҖҡиҝҮд»ҘдёӢжӯҘйӘӨдҪҝз”Ё HTTPS и®ҝй—® ElasticsearchгҖӮ

жӯҘйӘӨ 1пјҡжүҫеҲ°з”ҹжҲҗзҡ„иҜҒд№Ұ

еңЁе®үиЈ… Elasticsearch ж—¶пјҢзі»з»ҹдјҡжҸҗзӨәдҪ дҝқеӯҳд»ҘдёӢдҝЎжҒҜпјҡ

в‘ гҖҒCA иҜҒд№Ұи·Ҝеҫ„пјҡйҖҡеёёдҪҚдәҺ elasticsearch-8.10.0/config/certs/http_ca.crtгҖӮ

в‘ЎгҖҒ з”ЁжҲ·еҗҚе’ҢеҜҶз Ғпјҡй»ҳи®Өз”ЁжҲ·жҳҜ elasticпјҢеҜҶз ҒдјҡеңЁеҗҜеҠЁж—¶з”ҹжҲҗгҖӮ
